# Obsidian Plugin: Bookmark Line with Hotkeys

version: 1.0.0
author: "IvanSaldikov
authorUrl: https://github.com/IvanSaldikov/
fundingUrl: https://github.com/sponsors/IvanSaldikov

For desktops only.

### Inspired by PyCharm (JetBrains IDE)

What it does:
By using Ctrl + <1..9> set up the bookmarks in any line of a file and move across all of your bookmarks with these.

Right panel includes the list of the Bookmarks.
